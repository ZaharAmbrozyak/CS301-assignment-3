Це третє завдання з курсу Баз Даних
## Структура проекту
* task.md - опис завдань, які треба виконати
* tables.sql - скрипти для створення таблиць
* fill_tables.py - скрипт .py для генерації даних
* scripts.sql - основний файл, де було реалізовано функції, процедури, тригери та інші завдання.
 * answers.md - відповіді на теоретичні запитання
 * explain_analyze.md - пояснення запиту explain analyze
